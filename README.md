# Ascii-cam

## Features

- Limited character variations.
- Colorizing by HUE.
- HUE offset.
- HUE gain.
- Automatic resize to terminal size. (curses wrapping)

## Prerequisites

- numpy
- opencv-python
- python compiled with ncurses support

## Use

Run cam.py.
